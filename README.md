# 个人网站

## 项目简介
这是一个个人网站项目，采用现代化的响应式设计，为访问者提供良好的浏览体验。

## 技术栈
- HTML5
- CSS3
- JavaScript
- 响应式设计

## 主要功能
- 首页：公司简介和核心数据展示
- 关于我们：公司历史、核心价值观和主要成就
- 解决方案：各类技术解决方案展示
- 服务：技术支持、软件开发、咨询顾问等服务介绍
- 联系我们：联系方式和在线留言表单

## 项目结构
```
sambohrith.github.io/
│
├── index.html # 主页面
├── styles.css # 样式文件
├── script.js # JavaScript 文件
│
└── images/ # 图片资源目录
    ├── consulting.svg
    ├── logo.svg
    ├── software-dev.svg
    ├── tech-support.svg
    └── training.svg
```

## 部署
本项目使用 GitHub Pages 部署，访问地址：https://sambohrith.github.io/Lincoln-zhu

## 本地开发
1. 克隆项目 
   ```bash
   git clone https://github.com/sambohrith/Lincoln-zhu.git
   ```
2. 打开项目
   ```bash
   cd Lincoln-zhu
   ```
3. 使用浏览器打开 index.html 文件即可本地预览
